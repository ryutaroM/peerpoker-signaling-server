# PeerPoker Signaling Server

PeerJS signaling server for PeerPoker - a P2P planning poker application.

## Overview

This is a WebRTC signaling server that enables peer-to-peer connections between PeerPoker clients. It uses PeerJS to handle WebSocket connections and facilitate initial handshakes between peers.

## Features

- WebSocket-based signaling for WebRTC connections
- CORS enabled for cross-origin requests
- Lightweight and easy to deploy
- No data storage - pure relay server
